# Catalyst Chemistry
## العامل الحفاز لتعلّم الكيمياء

Catalyst Chemistry هو تطبيق عربي لتعلّم الكيمياء من المتصفح، مصمم لمساعدتك على **الفهم السريع** ثم **التدريب العملي** قبل الامتحان.

Catalyst Chemistry is an Arabic chemistry learning PWA built for **fast understanding** and **practical exam training**.

> **ملاحظة | Note:**
> هذا التطبيق **مجاني بالكامل**، وهو مشروع تعليمي مستقل **غير تابع** لوزارة التربية والتعليم.
> This app is **completely free** and is an independent educational project **not affiliated** with the Ministry of Education.

## لماذا Catalyst Chemistry؟ | Why Catalyst Chemistry?

**العربية**
- يجمع المذاكرة والتطبيق في مكان واحد بدل التنقل بين مصادر كثيرة.
- يربط القانون بالسؤال، وليس حفظ القوانين بشكل منفصل.
- يساعدك تركز على نقاط الضعف بخطة يومية تكيفية.

**English**
- Combines study and practice in one place.
- Connects formulas to actual questions, not isolated memorization.
- Focuses your revision on weak areas using an adaptive daily plan.

## أفضل استخدام للتطبيق | Best Use Flow

### خطة 20 دقيقة يوميًا | 20-Minute Daily Flow

**العربية**
1. ابدأ بـ **خطة اليوم** (3–5 دقائق).
2. راجع قانون أو فكرة من **القوانين/العضوي** (7 دقائق).
3. حل **اختبار قصير** (5 دقائق).
4. ثبّت الأخطاء عبر **البطاقات** (3 دقائق).

**English**
1. Start with **Today’s Plan** (3–5 min).
2. Review one concept in **Laws/Organic** (7 min).
3. Take a short **Quiz** (5 min).
4. Reinforce weak points with **Flashcards** (3 min).

### حسب الهدف | By Goal

| الهدف (العربية) | Best Path (English) |
|---|---|
| فهم درس جديد | Home → Organic/Lessons → Quick Quiz |
| حل مسائل بالقوانين | Laws → Apply formula → Quiz |
| مراجعة سريعة قبل الامتحان | Today Plan → Laws → Flashcards |
| تثبيت الجدول الدوري | Periodic Table → Element details → Quick recall |

## طريقة الاستخدام السريعة | Quick Start

**العربية**
1. افتح رابط التطبيق.
2. اختر القسم من الصفحة الرئيسية.
3. استخدم القوانين/الجدول/العضوي/الاختبارات حسب هدفك.
4. بعد أول تحميل، أجزاء كبيرة تعمل بدون إنترنت.

**English**
1. Open the app link.
2. Choose a section from the home screen.
3. Use laws/periodic/organic/quizzes based on your goal.
4. After first load, large parts work offline.
